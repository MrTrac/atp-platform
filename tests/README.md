# Tests

`tests/` chứa unit tests, integration tests, và fixtures tối thiểu cho ATP baseline hiện hành.

Phạm vi hiện hành:

- kiểm tra flow repo-local của ATP v0
- kiểm tra workspace boundary và foundational contract chain baselines v0.2–v1.0
- giữ integration tests dùng isolated workspace, không ghi vào runtime zone thật

Không thuộc scope:

- end-to-end orchestration qua remote plane
- performance/load testing
