# ATP Milestone Templates

## Mục đích

Bundle này cung cấp bộ template chuẩn cho milestone documents của ATP.

Nó dùng để giữ ổn định 4 lớp nội dung bắt buộc của mỗi version hoặc phase:

1. Proposal
2. Execution Plan
3. Freeze Decision Record
4. Closeout

## Khi nào dùng từng template

- `ATP_Milestone_Proposal_Template.md` - dùng khi mở một version hoặc phase mới
- `ATP_Milestone_Execution_Plan_Template.md` - dùng khi chuyển từ proposal sang execution planning
- `ATP_Milestone_Freeze_Decision_Record_Template.md` - dùng khi milestone đi tới freeze boundary
- `ATP_Milestone_Closeout_Template.md` - dùng khi milestone kết thúc và cần carry-forward rõ ràng

## Default 4-document ATP milestone model

ATP mặc định dùng 4-document model:

- Proposal
- Execution Plan
- Freeze Decision Record
- Closeout

Phase nhỏ có thể dùng merged documents, nhưng vẫn phải giữ đủ 4 logical content layers này.

## Naming convention cho real milestone files

- `ATP_v0_2_Proposal.md`
- `ATP_v0_2_Execution_Plan.md`
- `ATP_v0_2_Freeze_Decision_Record.md`
- `ATP_v0_2_Closeout.md`
