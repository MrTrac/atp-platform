## Tóm tắt scope

ATP v0.1 hardening focuses on:

- docs cleanup
- schema/model consistency
- CLI summary normalization
- inspect hardening
- fixture cleanup
- test coverage hardening
- artifact summary / final summary polish

ATP v0.1 hardening does **not** include:

- production workspace materialization
- approval UI
- remote orchestration plane
- advanced scheduling
- multi-provider arbitration
- new orchestration layers
